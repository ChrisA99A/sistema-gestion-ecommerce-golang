# sistema-gestion-ecommerce-golang
Sistema de gestión de e-commerce en Golang con enfoque modular, control de inventario, carrito de compras, pedidos y aplicación de programación funcional

## Módulos
- Usuarios
- Productos
- Inventario
- Carrito de compras
- Pedidos
- Pagos simulados
- Administración
- Reportes básicos

## Tecnologías
- Golang
- net/http
- encoding/json
- testing
- GitHub

## Ejecución
go run main.go

## Pruebas
go test ./...
